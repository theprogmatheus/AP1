## ▶️ Como Compilar e Executar

### 💻 Terminal (Linux/macOS)

```bash
gcc nome_do_arquivo.c -o programa
./programa
```


---
🪟 Windows (com MinGW)

```bash
gcc nome_do_arquivo.c -o programa.exe
./programa.exe
```
> Ou, se preferir, use uma IDE como Code::Blocks, VS Code (com extensão C/C++), ou Replit online.
